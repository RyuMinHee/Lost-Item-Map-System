## 작업 내용
이번 주차에는 분실물 지도 시스템의 초기 화면을 개선하고, 분실물 및 습득물 등록 기능의 기본 구조를 구현하였다.

## 구현 내용
- 분실물 / 습득물 구분 선택 기능 추가
- 물건 이름 입력 기능 추가
- 위치 입력 기능 추가
- 특징 입력 기능 추가
- 등록 버튼 클릭 시 목록에 항목이 추가되는 기능 구현
- JavaScript 파일(app.js) 분리

## 향후 개발 예정
- 지도 API 연동
- 등록 위치를 지도에 마커로 표시
- 검색 기능 구현
- 데이터베이스 연동
- 사용자 연락 기능 설계

## 비고
현재 구현된 기능은 프론트엔드 기반의 임시 등록 기능이며, 새로고침 시 데이터는 유지되지 않는다.
