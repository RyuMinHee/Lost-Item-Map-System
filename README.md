# Lost Item Map System

## 프로젝트명
분실물 지도 시스템

## 프로젝트 개요
분실물 지도 시스템은 사용자가 분실한 물건이나 발견한 물건의 정보를 지도 기반으로 등록하고 공유할 수 있는 웹 기반 시스템이다.

사용자는 분실 위치 또는 습득 위치를 지도에 표시하고, 물건의 종류, 특징, 시간 등의 정보를 등록할 수 있다. 이를 통해 분실물 정보를 빠르게 공유하고 물건을 찾을 가능성을 높이는 것을 목표로 한다.

## 주요 기능
- 분실물 등록 기능
- 습득물 등록 기능
- 지도 기반 분실물/습득물 조회 기능
- 물건 종류 및 위치 기반 검색 기능
- 사용자 간 연락 기능

## 개발 환경
- OS: Windows 10
- Language: HTML, CSS, JavaScript
- DB: MySQL 예정
- Map API: Kakao Map 또는 Google Map API 예정
- Version Control: GitHub

## 개발 방식
본 프로젝트는 Agile 방법론 중 SCRUM 모델을 기반으로 진행한다.
